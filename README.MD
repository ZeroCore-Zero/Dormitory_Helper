# Dormitory Helper

An assistant application provides some useful tools in dormitory.

## How to use

Backend:
> [!IMPORTANT]
> There are a number of bugs need to help, because this is an old version, and need to refactoring.

```bash
    cd ./backend
    pip install -r requirements.txt
    python app.py
```

Frontend:
```bash
    cd ./frontend
    npm install
    npm run dev
```
